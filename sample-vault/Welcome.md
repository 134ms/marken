# Welcome to Marken

Marken is a tiny, self-hosted **Markdown viewer**. Point it at a folder of `.md` files and start reading.

## What you get

- A clean three-pane reading layout on desktop
- Slide-in drawers for the tree and outline on mobile
- GitHub-flavored Markdown, including tables, task lists, and code blocks
- Math via $\KaTeX$ and diagrams via Mermaid
- Wiki-style links: see [[Markdown features]] for examples
- A fast full-text search box in the header (try pressing <kbd>/</kbd>)

## Getting around

The left side lists every document in your vault. The right side outlines the current page — click a heading to jump.

> Tip — on a phone, use the icons in the top bar to open the tree, the outline, or the **⋮** menu.

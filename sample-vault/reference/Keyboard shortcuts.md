# Keyboard shortcuts

| Action                       | Shortcut |
|------------------------------|----------|
| Focus the search box         | <kbd>/</kbd> |
| Move through search results  | <kbd>↑</kbd> / <kbd>↓</kbd> |
| Open the highlighted result  | <kbd>Enter</kbd> |
| Dismiss search / drawer      | <kbd>Esc</kbd> |
| Toggle theme                 | via the <kbd>⋮</kbd> menu |

## Browser conveniences

- Headings have anchor links — hover a heading to reveal the `#` and copy a deep link.
- Use the **⋮** menu to download the original `.md` source of the current page.
